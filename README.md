                    # lm13700_pcb-design
This is  a small one layer pcb design  on Kicad 10.0 ,of the LM13700  (which is an Operational Transconductance Amplificator)  using tht type components.

Due to the size  the roads may be  closed to each other which can make the hand soldering a little bit difficult, there is a pdf file which show the project from scratch to the 3d visualization . 

GERBER and DRILL folders are also given for the manufacturing.
